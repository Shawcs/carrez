

INSTALATION:


1. Ouvrez l'invite de commande et placer vous dans le fichier "./carez"

2. Faites "npm install" 

3. Puis "npm start" 

4. Ouvrez votre navigateur et entrez "localhost:3000"



Attention


Le server web n'accepte pas les url mal formées ainsi que certaines annonces comportants
des carractères spéciaux ainsi que des apostrophes dans les noms de villes.


les url suivantes fonctionnent:
un appartement : http://www.leboncoin.fr/ventes_immobilieres/903278558.htm?ca=12_s

une maison : http://www.leboncoin.fr/ventes_immobilieres/845672483.htm?ca=12_s
